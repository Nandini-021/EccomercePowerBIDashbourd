# 📊 Ecommerce Sales Dashboard | Power BI

## 📌 Project Overview

This project is an interactive **Ecommerce Sales Dashboard** developed using **Microsoft Power BI**. It provides a comprehensive analysis of sales performance, profitability, customer behavior, product categories, payment methods, and regional sales trends.

The dashboard enables business users to monitor key performance indicators (KPIs), identify high-performing products and states, analyze customer purchasing patterns, and make data-driven decisions.

---

## 🎯 Objectives

- Analyze overall ecommerce sales performance.
- Track revenue, profit, and average order value.
- Compare sales across different states.
- Understand customer purchasing behavior.
- Identify the most profitable product categories.
- Analyze preferred payment methods.
- Monitor monthly profit trends.

---

## 📈 Dashboard Features

### KPI Cards
- Total Sales Amount
- Total Profit
- Average Order Value (AOV)

### Interactive Filters
- State Filter
- Quarter Filter

### Visualizations
- Sales by State
- Quantity by Category
- Profit by Sub-Category
- Profit by Month
- Sales by Customer
- Quantity by Payment Mode

## 📂 Dataset Information

The dataset contains ecommerce transaction records including:

- Order Details
- Customer Information
- Product Category
- Product Sub-Category
- Sales Amount
- Quantity
- Profit
- Payment Mode
- State
- Customer name
- Order date/Order ID/State

---

## 📊 Key Insights

- Maharashtra generated the highest sales revenue.
- Clothing category contributed the largest sales volume.
- COD is the most preferred payment method.
- November recorded the highest monthly profit.
- Certain product sub-categories generated significantly higher profit margins.
